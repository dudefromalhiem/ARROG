Apply to Contribute to the Red Oaker Guild

Use this form to apply for a role in the Guild ladder: Contributor, Moderator, or Admin. Chief Admin and higher roles are appointment-only. Applications are reviewed manually by staff.

Contributor: create and edit your own submissions, upload media, and contribute archive entries.
Moderator: includes Contributor powers, plus submission review and report handling in moderation scope.
Admin: includes Moderator powers, plus application decisions, user management, and full report handling.
Chief Admin: appointment-only. Includes Admin powers with override authority for non-owner decisions.
