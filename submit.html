Workshop Guidebook

What this page does

Use this workshop to create a new page, update an existing page, or save a draft for later.

Choose a file type first, then fill in the title, page name, tags, and content.

How the page link works

The page link is the short address that appears after /pages/.

If you leave it blank, the workshop creates one from your title automatically.

Choosing a page type

  • Template Builder: easiest option for standard pages.
  • Document Studio: drag and arrange blocks like a document.
  • Code Editor: write your own HTML and CSS.

Before you publish

  • Make sure the title is clear and matches the page content.
  • Add images, audio, or video where needed.
  • Use drafts if you are not ready to publish yet.

Editing an existing page

  • Open the page from the admin area or a linked page card.
  • The workshop loads the current page so you can update it.
  • Save a draft first if you want to compare versions safely.

Saving and publishing

  • Drafts keep your work private until you are ready.
  • Publishing sends the page for review or posts it right away if you have higher clearance.
  • After publishing, the page opens at its public address.

Using media

  • Upload images for almost any page.
  • Use audio or video when the page type allows it.
  • Choose a file from the upload area, then pick it inside the page builder.

Finding your page later

  • Your public address starts with /pages/.
  • The history area shows everything you have submitted.
  • Drafts stay available until you remove them or publish them.
